# PIX GENERATOR

[![NodeJS](https://img.shields.io/badge/Node.JS-000?style=for-the-badge&logo=Node.js)](https://nodejs.org/en/)

## About
Generate QR codes for payments via PIX with this project made with Node.JS and Express.JS.
